Настройка Nginx и Firewall
Для запуска выполните команду:

ansible-playbook -i hosts.ini playbook.yml -K

Что сделает скрипт:

Установит Nginx и создаст страницу с информацией о сервере.

Настроит фаервол: доступ к сайту будет открыт только для вашей сети, для остальных — заблокирован.

Результат: Откройте IP-адрес сервера в браузере. Если всё успешно, загрузится страница с данными о системе.
